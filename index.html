WCSC is a new club to the Wentworth Comunity designed to build a community among the Wentworth students interested in learning the gift of programmming!

Hackathon

A hackathon is a gathering where programmers collaboratively code in an extreme manner over a short period of time. We hope to host a hackathon on Wentworths campus with the support and sponsorship of Major League Hacking

Higher Level Programming Languages

Rather than dealing with registers, memory addresses and call stacks, high-level languages deal with variables, arrays, objects, complex arithmetic or boolean expressions, subroutines and functions, loops, threads, locks, and other abstract computer science concepts, with a focus on usability over optimal program efficiency. We will use languages such as Java, C, and Python to enhance our skills as well as complete fun and interesting projects.

Web Development

We hope to have a few projects teaching and using web development programming such as HTML, CSS, and Java Script as well as backend servers and the steps it takes to create a functional website from start to finish.

Group Projects

Working in a group to tackle a coding task is a valuable skill a programmer can aquire. To make this task easier we will teach and use GitHub for most of these projects.

Mobile Development

Now a days it seems like there is an app for everything. We hope to learn and develope funtional apps for Apple's App Store and the Google Play Store using the software Swift and AndriodStudios.

Connections to Companies

Ocasionally our meetings will consist of talks from the leaders in the tech industry who will be happy to provide insight into their daily lives as well as share their wisdom of the company.
